## 7. List

- Unordered list 
  > Use -, *, + for unordered list <br>
  > -name <br>
  > -age <br>
  > *date <br>

    > Ex : 
    > - name
    > - age
    > - date

<br>

- Ordered list 
  > Use numbers <br>
  > 1.name <br>
  > 2.age <br>
  > 3.date <br>

   > Ex : 
   > Use numbers <br>
   > 1. name <br>
   > 2. age <br>
   > 3. date <br>

<br>

- Nasted List
    >Giving 2 space makes nasted list

  > - Fruits
  > - Apple
  > - Banana
  >   - Cavendish
  >   - Red Banana
  > - Vegetables
  >   1. Carrot
  >   2. Broccoli

<br>

- Task list (CheakBox)
    >-[ ] Learn Markdown <br>
    >-[x] Drink water <br>
    >-[ ] Submit assignment <br>

    >Ex :
    > - [ ] Learn Markdown
    > - [x] Drink water
    > - [ ] Submit assignment 

<br>
